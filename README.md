🎓 Student Performance Analysis Dashboard

📌 Project Overview

This project focuses on analyzing student performance using Python (Pandas) for data preprocessing and Power BI for interactive dashboard creation.

The dashboard provides insights into how factors like gender, parental education, and lunch type influence student scores.

📊 Dashboard Preview
https://github.com/Challasaiteja576/Student-Performance-analysis/blob/main/dashboard.png

🎯 Objectives

- Analyze student performance data
- Identify factors affecting academic scores
- Build an interactive and user-friendly dashboard
- Enable filtering for deeper insights

🛠️ Tools & Technologies

- Python (Pandas)
- Power BI
- CSV Dataset
- 
📂 Workflow

🔹 Data Processing (Pandas)

- Cleaned missing values
- Converted data types
- Removed inconsistencies
- Created structured dataset

🔹 Visualization (Power BI)

- Imported cleaned dataset

- Created interactive dashboard with:
  
   KPI Card (Total Average Score = 68K)
   Bar Chart – Average Score by Gender
   Bar Chart – Average Score by Parental Education
   Pie Chart – Average Score by Lunch Type
   Slicers – Gender & Parental Education

📈 Key Insights

- Female students show slightly higher average scores than male students
- Students with standard lunch perform better than those with free/reduced lunch
- Parental education level has a clear impact on student performance
- Students whose parents have higher education tend to score more

📌 Conclusion

This project demonstrates an end-to-end data analytics workflow — from raw data cleaning to creating meaningful visual insights using Power BI.
